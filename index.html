Every card has a title

This card has a paragraph of text with a link. It also has a second sentence of text with a link.

A card is a simple unit of information. We'll use the <article> tag to represent a card and then style it with CSS.

Every card has a width

But we never use pixels ("px") to determine the size of a card.

We use measurements that flex, like percents (%), or that relate to the size of the content, like root "em" units (rem), which relate the size of the element to the text size and thus makes the card very readable.

This card collects terms and their definitions into a Descritption List Element (dl) for later reference.

tag name
The word inside an HTML tag that defines the element.
tag
A tag name and opening and closing delimiters. A closing tag puts a forward slash in front of the tag name.
Element
An opening tag, some content, and a closing tag. Some elements don't have closing
html
it's a representational language, semantic markup, you use dumb things like article tags for google spiders
article tag
a unit of redistributable content
user agent
browser, usually. Actually I don't know what else it could be.
